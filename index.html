Hobbit Pirate is A decentralized meme token that evolved into a vibrant ecosystem. Hobbit Pirate with $HTP ticker, a Web3 meme token inspired by a fan-made mysterious of the crossover film between The Hobbit and The Pirates of Caribbean on BSC Network Backed by a strong ecosystem.

This film fans community are expected as HTP token holders together with cryptocurrency activists, especially meme token. As we said before that our HTP token is a meme token that has different level from the existing meme token. HTP token is not only as meme token, but also has more value where HTP token can make more benefits than just meme token.

Products

HTP token has features that other tokens do not have, and these features give opportunity for HTP token holders to make more benefits.

The features of Hobbit Pirates are:

Trade

Hobbit Pirate apart from being a meme token (HTP), it is also a decentralized exchange platform for exchanging BEP-20 tokens on BNB Chain Network.

Farming

In Hobbit Pirate Farm, there are two pairs, namely HTP/BNB LP HTP/BUSD with High APY depending on the chosen pairs.

Staking

In Hobbit Pirate, there are some ways used by the users to earn rewards: staking HTP tokens and staking NFT with APY 12% to earn HTP tokens.

Mystery Box

Users can earn mystery box as rewards by their purchased minting NFT. The Mystery Box they get contains random rewards depend on each user's luck.

Lottery

In Hobbit Pirate, there is also a feature called Lottery where users have opportunity to win lottery prizes in every certain period.

IFO Launchpad

The Hobbit Pirate was also the runway for the IFO Launchpad. IFO Launchpad helps DeFi projects to raise capital through farming features offered by the Hobbit Pirate decentralized exchange.
